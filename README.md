# D.Visual AI Studio

هذا المستودع يحتوي على موقع لتوليد شخصيات وصور باستخدام الذكاء الاصطناعي.  

### كيفية الاستخدام
1. افتح الموقع `index.html`.
2. اكتب برومت الشخصية في المربع الأول.
3. اكتب برومت الفيديو في المربع الثاني.
4. اضغط على زر "ابدأ الإنشاء".
5. ستظهر الصورة مباشرة داخل الموقع.

⚠️ ضع API Token الخاص بك في الكود كما هو موضح في `index.html`.
